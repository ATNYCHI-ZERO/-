## 3) Formal setup

Let (\mathcal{H}) be a finite-dimensional complex Hilbert space. A *cognitive state* is a unit vector (|\psi\rangle \in \mathcal{H}) (or density operator (\rho)). A binary decision ‚ÄúYes/No‚Äù to query (A) is represented by a projector (P_A) (with (I-P_A) for ‚ÄúNo‚Äù). The Born rule gives
[
\Pr(A{=}Y)=\langle \psi| P_A |\psi\rangle \quad \text{(or } \Pr(A{=}Y)=\mathrm{Tr}(\rho P_A)\text{).}
]

A *sequence* of questions (A,B) is modeled by sequential (generally non-commuting) projections. For outcomes (A{=}Y) then (B{=}Y):
[
\Pr(B{=}Y \ \text{after } A{=}Y)=
\frac{\langle \psi| P_A P_B P_A |\psi\rangle}{\langle \psi|P_A|\psi\rangle}.
]
If (P_A) and (P_B) **do not commute**, order effects emerge: (\Pr(A!\to!B)\neq \Pr(B!\to!A)). This is the core of the QQ-model that yields the QQ-equality empirically observed across surveys. ([PNAS][3])
